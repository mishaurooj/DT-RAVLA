
# DT-RAVLA Model Results

This directory contains all exported experimental results generated by the DT-RAVLA evaluation pipeline. The files are organized to support manuscript preparation, statistical analysis, reproducibility, and result verification.

The exported tables correspond directly to the experiments reported in the IEEE Transactions manuscript.

---

# Folder Contents

The directory contains four categories of outputs.

- Publication-ready LaTeX tables
- Mean ± standard deviation summaries
- Raw seed-wise evaluation results
- Supporting documentation

---

# Table Organization

| Table | Purpose |
|--------|----------|
| Table 2 | In-domain benchmark evaluation |
| Table 3 | Cross-process transfer evaluation |
| Table 4 | Robustness evaluation |
| Table 5 | Computational efficiency |
| Table 6 | Ablation study |

Each experiment is exported in multiple formats.

---

# File Types

## 1. LaTeX Tables

Files ending with

```
.tex
```

are ready for direct inclusion in the manuscript.

Example

```
Table_2_Benchmark.tex
Table_3_Cross_Process.tex
Table_4_Robustness.tex
Table_5_Efficiency.tex
Table_6_Ablation.tex
```

These tables already use IEEE formatting and can be inserted directly into the paper.

---

## 2. Mean ± Standard Deviation

Files ending with

```
Mean_Std.csv
```

contain aggregated statistics computed over all evaluation seeds.

Example

```
Table_2_Benchmark_Mean_Std.csv
```

Each row reports

- Mean
- Standard deviation

for every reported metric.

These files are intended for

- statistical comparison
- manuscript tables
- plotting
- significance analysis

---

## 3. Raw Seed Results

Files ending with

```
Raw_All_Seeds.csv
```

contain the complete results for every individual random seed.

Example

```
Table_4_Robustness_Raw_All_Seeds.csv
```

These files preserve the original measurements before averaging.

Typical columns include

- Seed
- Dataset
- Configuration
- Accuracy
- Balanced Accuracy
- Precision
- Recall
- Specificity
- F1
- MCC
- AUROC
- AUPRC
- Calibration metrics
- Threshold
- Runtime statistics

These files are useful when

- reproducing paper tables
- generating new figures
- computing confidence intervals
- performing additional statistical analysis

# Reproducing Results

To regenerate every table, run

```bash
cd Code

python run_all.py

python evaluate_pretrained_all.py

python evaluate_pretrained_consistent_subtables.py
```

All exported files will be recreated inside this directory.

---

# Notes

The LaTeX tables are generated automatically from the evaluation pipeline.

The CSV files should be treated as the authoritative source for numerical analysis.

If a discrepancy exists between a manually edited manuscript table and a CSV file, the CSV values should be considered the correct experimental output.
